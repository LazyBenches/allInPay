# all-in-pay
* 小应用（手机号码）注册（通联注册）->应用实名-小应用签约
## 流程
######必须流程
* 创建会员 1
* 设置企业信息 1
* 企业信息审核结果通知 1 (回调)
* 获取会员信息
* 发送短信验证码（企业会员需审核通过后才能绑定手机。） 1
* 绑定手机（企业会员需审核通过后才能绑定手机。）1
* 个人实名认证 1
* 签约 1
* 查询卡 bin 1
* 请求绑定银行卡 1
* 确认绑定银行卡
* 锁定会员
* 解锁会员
* 充值申请
* 重发支付短信验证码
* 提现申请
* 确认支付（提现后台+短信验证码确认）
* 确认支付（充值前台+短信验证码确认）
* 订单结果通知
* 查询订单状态
* 退款申请
* 平台转账
* 查询余额
* 通联通划款入账通知
* 平台账户集余额查询



######非必需流程给
* 提现设置安全卡
* 查询绑定银行卡
* 解绑绑定银行卡
